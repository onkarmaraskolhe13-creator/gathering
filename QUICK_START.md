## Get Started in 2 Minutes!

### Option 1: Use Right Away (No Installation Needed)

1. **Open the app:**
   - Open `index.html` in your web browser
   - That's it! The app is fully functional

2. **Create an account:**
   - Click "Sign Up"
   - Enter your name, email, and password
   - Click "Sign Up"

3. **Start using:**
   - Create posts with text, images, or links
   - Like and comment on posts
   - View and edit your profile
   - All data is saved automatically!

### Option 2: Use with Python Backend (Optional)

If you want a more robust setup with a database:

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Start the backend:**
   ```bash
   python app.py
   ```

3. **Open the app:**
   - Open `index.html` in your browser
   - Everything works the same, but data is stored on the server
